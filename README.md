# yolo-v3-by-tensorflow

> pip install -r requirements.txt


## Training model: [model.h5](https://drive.google.com/file/d/1s-yeFvDI54ixLgIbfbxGSFRpSGR_buWf/view?usp=sharing)

## inference: model.h5
